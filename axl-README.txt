
    json/axl

        JSON Auto XML

            * Map java classes to JSON+XML I/O DOM 

            * I/O XML for JSON+DOM tree

            * JSON+DOM tree usage is 100% transparent to (JSON|AXL) I/O


        XML display of attribute-properties is horizontal, while JSON
        display of attribute-properties is vertical.  JSON is
        convenient to smaller data sets, and XML is convenient to
        larger data sets.
